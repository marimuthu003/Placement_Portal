IF SOMETHING DOESN'T WORK
==========================

1. Check that requirements are installed:
   pip list

2. Check that you're in the right directory:
   pwd
   dir

3. Make sure venv is activated:
   echo $env:VIRTUAL_ENV

4. Check for syntax errors:
   python -m py_compile app.py

5. Try deleting placement.db and running again:
   rm instance/placement.db
   python app.py
